# 5. GitHub Releases

When you push an Annotated Tag to GitHub,
it automatically shows up in the "Tags" section. However,
a Release is a GitHub-specific feature that wraps around a tag.

- Release features:
  You can attach compiled binaries (like a .zip of your Node project),
  write detailed "Change Logs," and mark it as a "Pre-release."

> [!TIP]
> If you name your tag following Semantic Versioning (e.g., v1.2.3),
> GitHub’s UI makes it very easy to generate release notes automatically.

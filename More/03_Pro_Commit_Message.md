# Pro Commit Message

| Type     | When to use it                                         | Example                                |
| -------- | ------------------------------------------------------ | -------------------------------------- |
| feat     | A new feature for the user.                            | feat: allow users to upload avatars    |
| fix      | A bug fix.                                             | fix: resolve crash on database timeout |
| docs     | Documentation changes only.                            | docs: update README with API endpoints |
| refactor | Code changes that neither fix a bug nor add a feature. | refactor: simplify Prisma schema logic |
| chore    | "Updating build tasks, package manager configs, etc."  | chore: add nodemon to dev dependencies |

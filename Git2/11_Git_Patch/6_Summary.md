# Summary

| Command      | Action                                 | Best for...                                      |
| ------------ | -------------------------------------- | ------------------------------------------------ |
| format-patch | Creates a professional .patch file.    | Sharing formal commits with metadata.            |
| am           | Applies a .patch and creates a commit. | Receiving code from a teammate.                  |
| diff > file  | Dumps raw changes to a text file.      | "Quick sharing of ""work in progress."""         |
| apply        | Applies raw changes to files only.     | "Testing someone's ""idea"" without committing." |

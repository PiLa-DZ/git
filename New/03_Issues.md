# Issues --> To report a bug or suggest a feature

- Purpose ---> `To report (a bug) or (suggest a feature)`
- Content ---> `(Text, screenshots, logs)`
- Action ----> `A conversation about a problem`
- End Goal --> `Getting a task (assigned or closed)`
- Creating issues:
  - Issue titles and descriptions:
  - Assignees: users, who are then responsible for addressing the issue.
  - Labels: `(bug, enhancement, documentation, help wanted)` What kind of task is this?
  - States: `(Open, Closed, Pending)`
  - Comments: A conversation about a problem
  - Milestones: When should this be finished? (deadlines)
